# Wowl
Wowl is a basic chess engine written in C++.<br />

## Board representation
* 10x12 mailbox

## Search
* Iterative deepening with aspiration window
* Transposition table
* Quiescence search
* Delta pruning
* MVV-LVA
* SEE
* Killer moves
* History heuristic
* Null move pruning
* Futility pruning
* Late move reduction

## Evaluation
* Material evaluation with piece-square tables
* King safety
* Space and center control
* Mobility
* Pawn structure
* Passed pawns
